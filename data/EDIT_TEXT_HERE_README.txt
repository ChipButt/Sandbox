ASTRA editable text locations

General app/anomaly/inbox text:
- Edit data/content.js

Map page text, map image paths, pins, and overview zone boxes:
- Edit data/map-config.js
- Easiest method: use map-editor.html from the working/editor ZIP.

For GitHub updates, you usually only need to replace:
- data/content.js for app text changes
- data/map-config.js for map layout/text changes

Do not upload private printable scan targets or old QR folders to the public app.
